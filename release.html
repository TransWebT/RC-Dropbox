RC-Dropbox Release Notes/History

v1.1.0 (18-January-2012) - Feature Release
  • Added client-side column sorting of the file listing. Clicking any column header sorts the table by the contents of that column. Clicking the column header a second time reverses the order. Secondary sorts can be performed by shift-clicking more than one column.
  • Added Release Notes page.
  • By request, changed date display format in file listing to 12-hour time format.
v1.0.0 (16-August-2011) - Initial Release
  • Application source code posted to GitHub: https://github.com/TransWebT/RC-Dropbox.
